Cinema E-Booking Website

MongoDB, Express, React, Node.js (TypeScript), tRPC, Next.js, HTML, CSS

This website allows you to book movie tickets and reserve seats online. You can watch trailers, view showing times, and search for movies. The video showcases the functionality and privileges of normal users and system administrators. Users can purchase tickets while administrators can add movies, edit movie information and showtimes, add promotions, and suspend users. We used password salting and hashing to ensure the security of our users.
